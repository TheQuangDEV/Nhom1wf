# 🛒 PHẦN MỀM QUẢN LÝ CỬA HÀNG MINI  
(C# WinForms – SQL Server)

## 📌 Giới thiệu

Phần mềm Quản Lý Cửa Hàng Mini được xây dựng bằng C# WinForms kết hợp SQL Server nhằm hỗ trợ quản lý hoạt động bán hàng một cách nhanh chóng, chính xác và hiệu quả.

Hệ thống giúp quản lý:
- Sản phẩm
- Nhân viên
- Hóa đơn
- Khách hàng
- Doanh thu

---

## 🎯 Mục tiêu dự án

- Xây dựng ứng dụng quản lý bán hàng bằng C# WinForms
- Kết nối và thao tác dữ liệu với SQL Server
- Rèn luyện kỹ năng CRUD (Create, Read, Update, Delete)
- Làm việc nhóm và sử dụng GitHub quản lý mã nguồn

---

## 🛠 Công nghệ sử dụng

- Ngôn ngữ: C#
- Nền tảng: .NET Framework (WinForms)
- Cơ sở dữ liệu: SQL Server
- Công cụ quản lý mã nguồn: GitHub
- IDE: Visual Studio

---

## 📂 Chức năng chính

### 1️⃣ Quản lý sản phẩm
- Thêm, sửa, xóa sản phẩm
- Tìm kiếm sản phẩm
- Cập nhật số lượng tồn kho

### 2️⃣ Quản lý nhân viên
- Thêm, sửa, xóa nhân viên
- Phân quyền đăng nhập

### 3️⃣ Quản lý khách hàng
- Lưu thông tin khách hàng
- Tìm kiếm khách hàng

### 4️⃣ Quản lý hóa đơn
- Tạo hóa đơn bán hàng
- Tính tổng tiền tự động
- Lưu chi tiết hóa đơn
